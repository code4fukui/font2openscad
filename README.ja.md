# font2openscad

文字列をOpenSCADのpolygon配列に変換するツールです。TTF/OTFフォントを使用し、2D形状を生成できます。

## デモ
https://code4fukui.github.io/font2openscad/

## 機能
- TTF/OTFフォントを読み込み、任意の文字列を2D形状に変換
- OpenSCADのポリゴン定義コードを出力
- フォントサイズ、行間隔、文字間隔、曲線精度などのパラメータ調整が可能
- 生成したデータをPreview表示

## 必要環境
ウェブブラウザ

## 使い方
1. TTF/OTFフォントを選択または追加
2. 表示したい文字列を入力
3. 各種パラメータを調整
4. 「生成」ボタンをクリックし、出力されたOpenSCADコードをコピー
5. OpenSCADでポリゴンを3D化
