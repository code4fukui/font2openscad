# font2openscad

任意のTTF/OTFフォントから、テキストをOpenSCADの`polygon()`定義（ポイントとパス）に変換します。

## デモ

ブラウザでお試しください: **[https://code4fukui.github.io/font2openscad/](https://code4fukui.github.io/font2openscad/)**

## 動作の仕組み

このツールは、ブラウザ上でシンプルな2パネルのインターフェースを提供します:
- **入力パネル:** フォントを選択し、テキストを入力して、サイズ、字間、精度などのパラメータを微調整します。
- **出力パネル:** プレビューキャンバスに生成された2D形状が表示され、テキストエリアにはコピーしてすぐに使えるOpenSCADコードが出力されます。

## 機能

- **フォント対応:** ローカルの任意のTTFまたはOTFフォントファイルを使用できます。
- **組み込みフォント:** Noto Sans JPおよびIchigoJamフォントファミリーが標準で利用可能です。
- **出力のカスタマイズ:** パラメータを調整して最終的な形状を制御できます:
    - フォントサイズ（OpenSCAD単位）
    - 字間
    - ベースライン（Yオフセット）
    - 曲線分割（flatness）
- **パス合成:** 重なり合う文字のパスを自動的に結合し、単一のきれいなポリゴンにします。
