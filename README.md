# font2openscad

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Convert text to OpenSCAD polygon(points, paths).

## Demo
[Live demo](https://code4fukui.github.io/font2openscad/)

## Features
- Convert text to OpenSCAD polygon using TTF/OTF fonts
- Adjust font size, letter spacing, and curve precision
- Preview the generated OpenSCAD code
- Copy the output code to clipboard

## Usage
1. Select a TTF/OTF font file or choose from the predefined options.
2. Enter the text you want to convert.
3. Adjust the font size, letter spacing, and curve precision as needed.
4. Click the "生成" (Generate) button to see the preview.
5. Copy the output code from the textarea to use in your OpenSCAD project.
